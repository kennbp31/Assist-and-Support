Assist and Support

Usage This application is designed to allow individuals with limited mobility and limited motor coordination to quickly and easily make phone and video calls using Skype. THIS APPLICATION IS NOT A REPLACEMENT FOR AN EMERGENCY CALL BUTTON!

Options > Add/Edit Contacts: This allows you to add in your contacts. Marking a contact as a favorite will add them to the homescreen.
Options > Preferences: This option allows you to setup your input device. This application has been designed to work with user defined keypresses, adaptive equipment such as Puff/Sip Devices, OR buttons/gamepads.
