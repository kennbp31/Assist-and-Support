Skype Call Delay

Less Delay More Delay


This setting determines how long the computer pauses to wait for Skype to load. Typically if you have a newer/fast computer you will want to set the delay low. If you have an older computer it is recommend to increase the delay. An increased delay will give your computer time to finish loading Skype before starting the call. If there is not enough load time for your computer, Skype calls WILL NOT WORK!.
